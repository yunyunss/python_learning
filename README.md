## 변수
변수는 변수명 = 표현식 이런식으로 선언함

이때 표현식은

"표현식(expression) : 값, 변수, 연산자의 조합. 값이나 변수 그자체로 표현식으로 취급되기도 한다."

```python
a = 1
b = '1abc'
d = "Abcd"
e = True
d = False
```

## 연산자
### 산술연산
더하기: +
빼기: -
곱하기: *
나누기: /
몫: //
나머지: %
제곱: **

```python
print(1 + 1)
#출력은 2
print(1 - 1)
#출력은 0
print(1 * 2)
#출력은 2
print(2 / 1.4)
#출력은 1.4285714285714286
print(2 // 1.4)
#출력은 1
print(3 % 2)
#출력은 1
print(3**2)
#출력은 9
```

### 문자열 연산
문자열에도 연산자를 사용할 수 있음

```python
print("Hello" + "Goodbye") #이어붙이기
#출력은 HelloGoodbye
print("Hi" * 3) #곱한 만큼 문자열을 반복
#출력은 HiHiHi
```
